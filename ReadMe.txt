②masterブランチに置いてあるものが利用可能なもの。マップエディタ完成版と同じように、編集モードの実装
(地形の上げ下げ、種類の変更)をした。

③Stageクラスでブロックを操作する際、for文で書いているため、クリックした直線上にある中で配列の1番若い場所が
操作されてしまい、裏からクリックすると手前側が上がってしまうため、そうならないように処理を追加した。
申し訳程度に上昇下降、位置リセットの操作を入れた。

④押しているブロックでない場所が動く現象に悩まされた。現状分かっているのは、
・回転、移動することでブロックの当たり判定がおかしくなって(歪んで)いく
・Stageのマウス座標変換は大丈夫なように見える（座標変換について理解できていない部分が多い
ため、問題個所を見つけれていない可能性）
※映像を見るとマウスカーソルが派手にズレているけど、録画した時はブロックの位置を押していたので気にしないでください

構想(実装したかったけど間に合いそうになくて断念したやつ。今後のために書き残す)
・「種類の変更」モードからの派生。現在の仕様ではクリックした列全てが変わるが、クリックしたY列のブロックだけ種類を
変更するモードを既存の物と切り替えられる形で実装
・範囲選択モード。名前の通り、編集するブロックを①クリック押しっぱなし②"①"でクリックした場所を始点、放した場所を
終点として四角形の範囲を生成③編集モードごとで指定した範囲全てのブロックを操作
・ブロックプレビュー。今カーソルを置いてる場所にブロックの設置予測みたいな感じで半透明のブロックを表示させる

～操作方法～
WASD	→　視点を前後左右に移動
矢印キー	→　視点を回転
Space	→　カメラ位置を上昇
LControl→　カメラ位置を下降
R	→　カメラ位置を初期状態に戻す
